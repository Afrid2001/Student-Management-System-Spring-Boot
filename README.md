# Student Management System - Spring Boot

A modern Student Management System built using Spring Boot, Thymeleaf, Bootstrap, and MySQL. This application helps administrators manage student records efficiently through a secure dashboard with student profile management, image uploads, PDF generation, and search functionality.

## Features

### Authentication & Security

* Secure Login System
* Change Password Functionality
* Session-Based Authentication
* Protected Routes for Authorized Users

### Student Management

* Add New Student
* View Student Details
* Update Existing Student Information
* Delete Student Records
* Search Students by Keyword

### Student Profile Features

* Upload Student Profile Photo
* Display Student Images in Student List
* View Student Information with Profile Photo
* External Image Storage Support

### PDF Export

* Generate Individual Student PDF Reports
* Export Student Details as PDF
* Include Student Profile Photo in PDF
* Dynamic PDF Naming Based on Student Information

### Dashboard

* Modern Dashboard Interface
* Student Statistics Overview
* Quick Access Navigation Cards
* Responsive Bootstrap Design

### User Experience

* Success Notifications for Operations
* Clean and Responsive UI
* Easy Navigation Between Pages
* Professional Layout and Design
